# AI Resume Analyzer

## Features
- Upload Resume (PDF)
- Job Description Matching
- ATS Score Prediction
- Keyword Analysis
- Resume Suggestions

## Tech Stack
- Python
- Streamlit
- Scikit-learn
- NLTK
- PyPDF2
- Matplotlib

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
```
